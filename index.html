OrangeSlice Teacher Rubric and Student Rubric

Teacher Rubric and Student Rubric are excellent tools for rubric grading in the classroom. This page briefly introduces the add-ons, states what permissions are requested and how they are used, details the privacy and security policies and lastly, the service terms.

Teacher Rubric

User Functionality Enhancements

Easily and quickly make rubric selections and convert the Analysis or Holistic rubric into a percentage or points grade. Teacher Rubric is an add-on that will increase your grading productivity and it professionally presents a scored rubric and grade for your students.

Google docs and Classroom are fantastic applications to create, distribute and receive submitted assignments from students. However, the grading process seems to bring the efficiency to a screeching halt. Where can the grade be placed so it stands out from the assignment? How can rubric selections be made easily? I like rubrics but the final grade can be tedious to calculate. It takes too many highlights, clicks and time to format grades on 125 submissions.

OrangeSlice: Teacher Rubric increases the teacher’s grading productivity by eliminating repetitive clicks, presenting the rubric selections in an easy to use format and presenting the final grades in a consistent, professional format. More time is created for the teacher to provide the needed constructive feedback their students need for success.

Documents Access

This scope request provides two functions for the user. The first gives permission for the add-on to color in the rubric. Both the Student and the Teacher benefit from this functionality. The performance levels achieved per category are clearly identified.

The permission also allows student feedback to be transcribed from the add-on to the doc. The teacher can enter feedback for each category and can write an overall summary of the student’s work. This provides an explanation of the performance levels reached by the student. The teacher can also provide encouraging and constructive feedback to improve their future school work.

Drive Access

This access allows the add-ons to rename the doc and provides access to the directory of files and folders within Google Drive. The renamed doc and list of folders are used to decrease the teacher’s grading time and find assignments that are ready to be graded, quickly see the grades for a whole class and to identify students who have not turned in an assignment.

The add-ons will rename the doc after it has been self-reviewed, peer-reviewed, returned or graded. The doc filename is modified to capture the review/graded status and includes the assignment’s grade. This benefits the student and the teacher as the filename immediately informs the viewer as to the assignments status. The teacher has an added benefit, managing assignment grading. When the assignments are viewed in Google Drive, the teacher has a roster list of students and their graded status is clearly marked by the filename. The teacher can select just those assignments that need to be graded. For those having completed the assignment, the teacher has an alphabetized list of grades received.

Script Container Access

This access allows the add-on to be added to your list of installed add-ons and allows the creation of a sidebar where the add-on is presented. From the menu, the add-on can be opened, the rubric can be processed with the scored grades and the add-on can be opened in different languages.

Student Rubric

User Functionality Enhancements

This is the student version of Teacher Rubric. Students are able to review their peer's work, make rubric selections and provide written feedback. The complete bundle of feedback is then presented to the student with a professional appearance.

The peer's review becomes a permanent record on the student's assignment. Once the teacher uses Teacher Rubric to make the final assessment, the peer's comments and rubric feedback are maintained. This allows the student to see their progression from early on until the end. In fact, the teacher can deepen the student's journey by doing a spot check, holistic review of the student's work. At the conclusion of the assignment, there would be record of three reviews: a peer's review, a teacher's holistic review and finally the teacher's final grade. This historical record can then be the starting point student reflections and teacher-student conversations on how to improve.

User Privacy

Teacher and Student data is only accessed for the purpose of the add-on’s intent; such as, the student’s name is placed in the doc’s title after grading or the Teacher’s drive is accessed to determine the number of students who have not yet completed the assignment. User information is not collected or stored. Student information is not collected or stored.

User data is never collected nor shared with a marketing company.

Actionable event data or click data is sent to Google Analytics with the sole purpose of improving the user experience within the add-ons.

Data Security

All OrangeSlice add-ons process data as provided by Google and never does the code give access outside of the Google domain. On the client side, the add-ons run self contained within the web page. There are necessary calls to the Google server to process information and those are completed with the code structure Google provides. The operating functions of the add-ons never goes beyond those definitions provided by Google. Code does not exist in the add-ons that references, calls upon or writes to table or database outside of the Google domain.

Actionable event data or click data is sent to Google Analytics with the sole purpose of improving the user experience within the add-ons. This data is sent to Google Analytics using code specified by Google. The code does not call upon a third party to collect or process the click data.
